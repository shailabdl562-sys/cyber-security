The Principles of Cyber Security

Cyber Security protects systems, networks, and data from digital attacks that aim to access, change, or destroy sensitive information.

1. Protecting Personal Data

Cyber security ensures personal information such as addresses, social IDs, and credentials remain safe from identity theft.

2. Securing Online Transactions

It keeps your online banking, payments, and shopping safe through encryption and secure protocols.

3. Preventing Unauthorized Access

Cyber security blocks attackers from gaining access to your devices, networks, and accounts.

4. Safeguarding Businesses

Organizations use strong firewalls, authentication, and monitoring to protect critical data from hackers.

5. Building a Safer Digital World

Cyber security shapes the future by creating a safer and more resilient online environment for all.

def encrypt(text, shift):
    result = ""
    for char in text:
        if char.isalpha():
            base = 65 if char.isupper() else 97
            result += chr((ord(char) - base + shift) % 26 + base)
        else:
            result += char
    return result

def is_safe_input(user_input):
    keywords = [";", "--", "DROP", "DELETE", "INSERT", "UPDATE"]
    for word in keywords:
        if word.lower() in user_input.lower():
            return False
    return True
